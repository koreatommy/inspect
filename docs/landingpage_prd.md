# 2. 문제 제기 섹션

## 아직도 엑셀, 종이대장, 수기 서명으로 관리하고 계신가요?

어린이놀이시설 안전점검 업무는 단순한 기록 업무가 아닙니다.
법적 의무사항을 놓치지 않아야 하고, 점검 결과를 기록·보관해야 하며, 필요 시 지도점검이나 민원 대응 자료로 즉시 제출할 수 있어야 합니다.

### 현장에서 자주 발생하는 문제

| 기존 관리 방식       | 발생 문제                        |
| -------------- | ---------------------------- |
| 종이 점검대장 작성     | 분실, 누락, 보관 불편                |
| 엑셀 수기 관리       | 시설 수가 늘어나면 이력 추적 어려움         |
| 사진 별도 저장       | 어떤 시설, 어떤 기구의 사진인지 확인 어려움    |
| 안전관리자 서명 수기 처리 | 위탁점검자·관리자 확인 절차 비효율          |
| 법적 의무사항 별도 확인  | 설치검사, 정기검사, 교육, 보험 만료일 누락 위험 |
| 보고서 수작업 작성     | PDF 변환, 제출용 문서 정리에 시간 소요     |

# 3. 서비스 핵심 메시지

## 점검은 모바일로, 관리는 대시보드로, 보고서는 PDF로

어린이놀이시설 안전점검 SaaS는 시설별 안전점검 데이터를 체계적으로 기록하고, 관리자가 언제든지 점검 현황과 법적 의무사항을 확인할 수 있도록 지원하는 안전관리 플랫폼입니다.

### 핵심 가치

1. **월 1회 안전점검 업무 표준화**
   시설별 점검 항목을 디지털화하여 누락 없는 점검을 지원합니다.

2. **법적 의무사항 확인**
   설치검사, 정기시설검사, 안전교육, 보험가입 등 주요 관리 항목을 한눈에 확인할 수 있습니다.

3. **점검대장 자동 생성**
   입력한 점검 결과를 기반으로 월점검 대장을 PDF로 생성하고 미리보기·다운로드할 수 있습니다.

4. **현장 사진과 위해요인 기록**
   기구별 위해요소, 수리 전·후 사진, 조치 이력을 시설 단위로 관리합니다.

5. **통계 기반 관리**
   양호, 요주의, 요수리, 이용금지 상태를 월별로 분석하여 시설 관리 우선순위를 판단할 수 있습니다.

관리자는 별도의 문서 편집 없이 제출 가능한 형태의 보고서를 확보할 수 있습니다.

소규모 시설은 Starter로 시작하고, 시설 수가 증가하면 Growth 또는 Pro로 확장할 수 있습니다.
Growth와 Pro는 관리 개소 수에 따라 단가가 낮아지는 누진 구조를 적용합니다. ([Safeplay][1])

네. 다수 시설을 관리하는 지자체, 교육청, 공공기관은 Growth 또는 Pro 요금제가 적합합니다. Pro에서는 API 연동 지원과 전담 고객 성공 매니저 기능까지 구성할 수 있습니다. ([Safeplay][1])
